# 생년월일 컬럼 매핑(columns + row[birthDateColumn]) 완성

- 날짜/시간: 2026-01-24 20:02:02
- 작업 폴더: `/Users/dev/onlyT_test1`

## 사용자 프롬프트

생년월일 컬럼 매핑 기능을 완성한다.

요구사항:
1. 엑셀 업로드 시 모든 컬럼명을 columns 배열로 프론트에 전달한다.
2. 생년월일 select dropdown을 columns 목록으로 채운다.
3. 선택된 컬럼명을 birthDateColumn state로 저장한다.
4. 테이블 row에서 row[birthDateColumn] 값을 생년월일로 표시한다.
5. 날짜 포맷을 YYYY-MM-DD로 변환한다.
6. 주민등록번호 컬럼 매핑 방식과 동일한 구조로 구현한다.

## AI 답변

- 백엔드
  - `POST /student-records/upload-excel` 응답에 `columns`(업로드된 모든 엑셀 컬럼명) 배열을 추가.
  - 각 학생 row 객체에 “엑셀 원본 컬럼 값”을 함께 합쳐 내려서 프론트에서 `row[birthDateColumn]` 접근이 가능하도록 처리.

- 프론트
  - `columns`로 생년월일 헤더 dropdown 옵션을 구성.
  - 선택값을 `birthDateColumn` state에 저장.
  - 셀 표시값은 `formatBirthDate(row[birthDateColumn])`로 `YYYY-MM-DD`로 정규화해 텍스트 표시.
